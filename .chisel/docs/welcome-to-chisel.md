Chisel is a local-first knowledge base for your project. Everything you see here is stored as plain Markdown files in your project.

### Quick Start
- Use **Docs** to manage your project's permanent knowledge base.
- Use **Specs** (`chisel spec`) to manage lifecycle-aware feature specs.
- Use `chisel context create <query>` to generate LLM context from docs and specs.

### Navigation
- `Tab`: Switch between Category, Document, and Preview panes.
- `j`/`k`: Move selection up and down.
- `q`: Exit the TUI and return to your shell.
